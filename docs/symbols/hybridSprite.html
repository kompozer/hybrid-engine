Function Namespace hybridSprite

The sprite object represents an active sprite during the game. All active sprites are handled in an array by the resource manager. They are generated during the game from the prototype templates through events. They are also removed from the game whenever they 'die', i.e. the number of active sprites always changes.
Defined in: </home/nils/repository/hybridjs/src/hybridSprite.js>.
