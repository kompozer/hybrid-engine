Namespace resourceManager

The resource manager is a global object that is created on game engine initialization. It handles all the resources that are needed by the gameplay, i.e. active sprites, tiles, sounds, events and so on. The resource manager is initialized by the game loop.
Defined in: </home/nils/repository/hybridjs/src/hybridResourceManager.js>.
