Function Namespace hybridMove

This module handles the viewport and sprite movement and the consequences. It performes a image (or pixel) collision detection and calls the respective sprite's impact and damage functions. It also handles sprite destruction and the case when a sprite leaves the game border, i.e. removal from the game.
