Function Namespace hybridEvent

Represents a single event that can occur during the game. Events are triggered through trigger Objects which hold the activation conditions. There are events for adding sprites, animations and soundeffects
Defined in: </home/nils/repository/hybridjs/src/hybridTrigger.js>.
